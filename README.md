# Bolum Pusulasi

Odev dosyasi: [ODEV.md](/home/drvoid/Desktop/HelloAI/ODEV.md)

Bolum Pusulasi, universite tercihi yaparken **yazilim, siber guvenlik, donanim/gomulu sistemler ve elektronik** alanlari arasinda daha bilincli karar vermeyi kolaylastiran modern bir web uygulamasidir.

Proje; tek bir test sonucuna dayanmak yerine kullaniciyi alanlari tanimaya, mini denemeler yapmaya, universiteleri karsilastirmaya ve kendi karar surecini yazili hale getirmeye yonlendirir.

## Ozellikler

- **Alan testi:** Kullanici cevaplarina gore yazilim, siber guvenlik, donanim/gomulu ve elektronik skorlarini hesaplar.
- **Sonuc analizi:** En yakin alanlari, guclu yonleri ve dikkat edilmesi gereken noktalari gosterir.
- **Alan rehberi:** Her alanin gunluk is tarzi, uygun bolumleri, avantajlari ve zorlayici taraflarini sade sekilde aciklar.
- **Mini proje gorevleri:** Kullaniciya alanlari deneyerek tanimasi icin kisa ve uygulanabilir gorevler sunar.
- **Universite karsilastirma:** Program, sehir, basari sirasi, ders plani, lab, kulup ve staj notlarini karsilastirmaya yardim eder.
- **Tercih listesi:** Aday bolumleri kisa listeye ekleme, siralama ve not alma imkani verir.
- **Karar gunlugu:** Kullanici karar surecini, kaygilarini, motivasyonlarini ve sonraki adimini kaydedebilir.
- **TR / EN dil destegi:** Arayuz Turkce ve Ingilizce olarak kullanilabilir.
- **Gece / gunduz modu:** Tema tercihi tarayicida saklanir.
- **Responsive tasarim:** Telefon, tablet ve masaustu ekranlarda kullanima uygundur.
- **Yerel veri saklama:** Ilk surumda veriler backend olmadan tarayicinin `localStorage` alaninda tutulur.

## Teknolojiler

- React
- Vite
- Tailwind CSS
- Lucide React
- localStorage

## Kurulum

Projeyi yerelde calistirmak icin:

```bash
npm install
npm run dev
```

Uretim build'i almak icin:

```bash
npm run build
```

Build sonucunu onizlemek icin:

```bash
npm run preview
```

## Katki

Katkilar memnuniyetle kabul edilir. Yeni test sorulari, daha iyi aciklamalar, kaynak guncellemeleri, tasarim iyilestirmeleri, hata duzeltmeleri ve yeni ozellik onerileri projeyi daha faydali hale getirir.

Katkida bulunmak icin bir issue acabilir, onerilerinizi paylasabilir veya pull request gonderebilirsiniz. Simdiden emek veren herkese tesekkurler.
