# Workshop Bilgi Notlari (Knowledge Base)

Bu klasor, bilgisayar bolumlerine yonelik verilen atolye/workshop boyunca islenen konularin duzenli notlarini icerir. Her konu ayri bir `.md` dosyasinda detaylandirilmistir.

## Icindekiler

1. [Github ve Git](01-github-ve-git.md) — Github hesabi, git kullanimi, branch/teamwork/changelog, fake commit'in tehlikeleri.
2. [FrontEnd vs BackEnd, LLM Chat vs Agentic AI](02-frontend-backend-ve-agentic-ai.md) — temel web mimarisi ve AI agent kavrami.
3. [Problem Cozme Sureci](03-problem-cozme-sureci.md) — fikirden 0-100 uygulama planina, checklist'e ve stabil commit/push akisina kadar 7 adimlik yontem.
4. [Open Source Katki Sureci](04-open-source-katki-sureci.md) — fork, PR, merge terimleri ve acik kaynak projeye katki adimlari.
5. [Agent Ciktilarini Denetleme](05-agent-ciktilarini-denetleme.md) — live test, logic test, unit test vs real test.
6. [Dokumantasyon](06-dokumantasyon.md) — `docs/*.md` kullanimi, formati ve neden onemli oldugu.
7. [Agent Kurallari ve Araclari](07-agent-kurallari-ve-araclari.md) — Codex/Claude/VSCode Chat, `AGENTS.md`/`GEMINI.md`/`CLAUDE.md`, reasoning, mini modeller, extra thinking, skills/plugins/tools.
8. [IDE ve Kurulum](08-ide-ve-kurulum.md) — IDE kavrami, VSCode'un tercih sebebi, Codex/Claude eklenti kurulumu, global git ayarlari.
9. [Git Repo Temelleri](09-git-repo-temelleri.md) — README kullanimi, badge'ler, lisanslar.
10. [Web Temelleri ve Guvenlik](10-web-temelleri-ve-guvenlik.md) — website odevi, HTTPS vs HTTP, guest wifi guvenligi ve loglama.
11. [Siber Guvenlik Icin Temel Bilgi](11-siber-guvenlik-icin-temel-bilgi.md) — algoritma ve yazilim mantiginin siber guvenlik icin neden sart oldugu.
