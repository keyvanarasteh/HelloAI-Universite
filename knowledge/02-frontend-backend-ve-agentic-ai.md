# FrontEnd vs BackEnd, LLM Chat vs Agentic AI

## FrontEnd vs BackEnd

- **FrontEnd:** Kullanicinin tarayicida gordugu ve etkilesime gectigi kisim (HTML, CSS, JS, React gibi arayuz teknolojileri).
- **BackEnd:** Verinin islendigi, saklandigi ve is mantiginin calistigi sunucu tarafi (API'ler, veritabani, sunucu mantigi).
- Bir web uygulamasi genelde bu iki katmanin birlikte calismasiyla olusur; static bir site sadece FrontEnd icerebilir (bu projede oldugu gibi).

## LLM Chat vs Agentic AI

- **LLM Chat:** Kullanicinin soru sordugu, modelin metinle cevap verdigi klasik sohbet arayuzu (ornek: normal bir chatbot).
- **Agentic AI:** Modelin sadece cevap vermekle kalmayip, arac (tool) kullanarak dosya okuma/yazma, komut calistirma, arama yapma gibi eylemleri kendi basina zincirleme sekilde gerceklestirebildigi sistemler (ornek: Claude Code, Codex).
- Fark: Chat modeli "soyler", agent "yapar" — gorevi adim adim planlayip araclarla gerceklestirir.
